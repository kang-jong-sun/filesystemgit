### 전략B 설계 근거

```
문제: 전략A의 EMA(250)/EMA(1575)는 느려서 횡보장에서 SL 반복
     → 6년간 SL 184건 = -$287M 손실

해결: 빠른 EMA(9)/EMA(100)로 추가 수익 기회 포착
     → 단, EMA 갭 < 0.5% (밀착/횡보)에서는 스킵
     → 갭 ≥ 0.5%일 때만 진입 (94,710건 분석 결과)

REV 제거 이유: 빠른 EMA는 크로스가 자주 → REV가 소액 손실 반복
TSL 설정: TA 5% 도달 시 TSL 활성화 → 0.3% 트레일 (빠른 수익 보존)
```
