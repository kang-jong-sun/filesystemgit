## 4. 전략 구조

### 4.1 진입 로직

```
1. EMA Cross 감지:
   golden_cross = EMA(250)[i] > EMA(1575)[i] AND EMA(250)[i-1] <= EMA(1575)[i-1]
   death_cross  = EMA(250)[i] < EMA(1575)[i] AND EMA(250)[i-1] >= EMA(1575)[i-1]

2. 감시 시작:
   golden_cross → watching = LONG, watch_bar = i
   death_cross  → watching = SHORT, watch_bar = i

3. 진입 조건 (감시 후 1봉 이상):
   IF i > watch_bar AND i - watch_bar <= MONITOR_WINDOW(6):
     IF capital >= $1,000:
       ENTER position (LONG or SHORT)
       position_size = $10,000 (margin $1K x 10x)
       set SL at entry_price * (1 ∓ 2.0%)
```

### 4.2 청산 로직

```
우선순위 1: SL (Stop Loss)
  LONG:  low <= SL_price → 청산 at SL_price
  SHORT: high >= SL_price → 청산 at SL_price

우선순위 2: TSL (Trailing Stop)
  활성화 조건: 미실현 수익 >= +60% (TA_PCT)
  활성화 후:
    LONG:  highest_high 갱신 → SL = highest * (1 - 8%)
    SHORT: lowest_low 갱신 → SL = lowest * (1 + 8%)
  체결: 현재가가 trailing SL 도달 시 청산

우선순위 3: REV (Reversal)
  반대 방향 MA cross 발생 시 현재가로 청산
```

### 4.3 수수료 계산

```
진입 시: capital -= position_size * FEE_RATE ($10,000 * 0.0004 = $4)
청산 시: PnL = (exit - entry) / entry * position_size * direction - position_size * FEE_RATE
```
