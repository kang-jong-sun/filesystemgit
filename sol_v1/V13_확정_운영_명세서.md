### 1.2 Confidence Score 계산 (포지션 사이징용)

```python
def calculate_score(adx, rsi, lr_slope, adx_rising, skip_diff_dir, vol_ratio):
    score = 0

    # ADX 강도 (0-25)
    if adx >= 30:        score += 25
    elif adx >= 25:      score += 18
    elif adx >= 22:      score += 12

    # RSI 위치 (0-20)
    if 40 <= rsi <= 60:  score += 20
    elif 35 <= rsi <= 65: score += 15
    elif 30 <= rsi <= 65: score += 10

    # LR_slope 안정성 (0-15)
    abs_slope = abs(lr_slope)
    if abs_slope < 0.3:  score += 15
    elif abs_slope < 0.5: score += 10

    # ADX 상승 중 (0-15)
    if adx_rising:       score += 15

    # Skip Same Direction (0-10)
    if skip_diff_dir:    score += 10

    # Volume Ratio (0-15)
    if vol_ratio >= 1.5: score += 15
    elif vol_ratio >= 1.2: score += 10

    return score  # 0~100
```

        partial_price = entry_price * 1.05
        close_partial(partial_size, partial_price)
        partial_done = True
        # 나머지 70% 보유 → 큰 추세 노림

# 4. TA Activation (Trailing Stop 활성화)

    sl_price = max(sl_price, new_stop)  # 위로만 이동
    if low <= sl_price:
        close_all_at(sl_price)

# 6. REV (Reversal) - TSL 미활성 시만
if not TSL_active and ema_cross_down:  # LONG에서 cross down
