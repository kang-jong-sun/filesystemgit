# 🚀 SOL Futures 자동매매 시스템 v4.0

## 📋 개요
Quadruple EMA (10_close/21_open/21_high/21_low) 터치 전략을 사용하는 SOL/USDT 선물 자동매매 시스템입니다.
AI 없이 순수 기술적 지표만으로 작동합니다.

## 🔴 주요 특징

### 전략
- **메인 지표**: Quadruple EMA Touch Strategy
  - EMA10_Open: 시가(Open) 기준
  - EMA21_Open: 시가(Open) 기준
  - EMA21_HIGH: 고가(High) 기준
  - EMA21_LOW: 저가(Low) 기준
- **타임프레임**: 
  - 메인: 30분봉 (진입/청산 신호)
  - 보조: 1m, 3m, 5m, 15m (신호 필터링)

### 📈 매수(Long) 포지션 진입 조건

**조건 1 - 근접 터치:**
- EMA10_open > EMA21_open (정배열)
- EMA21_high의 0.2% 이내 근접
- 3분(180초) 이상 유지
- 1m/3m/5m/15m 중 2개 이상 정배열 3분 이상 유지

**조건 2 - 상향 돌파:**
- EMA10_open > EMA21_open (정배열)
- EMA10_open > EMA21_high
- 3분(180초) 이상 유지
- 1m/3m/5m/15m 중 2개 이상 정배열 3분 이상 유지

### 📉 매도(Short) 포지션 진입 조건

**조건 1 - 근접 터치:**
- EMA10_open < EMA21_open (역배열)
- EMA21_low의 0.2% 이내 근접
- 3분(180초) 이상 유지
- 1m/3m/5m/15m 중 2개 이상 역배열 3분 이상 유지

**조건 2 - 하향 돌파:**
- EMA10_open < EMA21_open (역배열)
- EMA10_open < EMA21_low
- 3분(180초) 이상 유지
- 1m/3m/5m/15m 중 2개 이상 역배열 3분 이상 유지

### 포지션 관리
- **마진 모드**: 격리마진 (ISOLATED)
- **포지션 크기**: 계정 잔액의 20%
- **레버리지**: 10배
- **보유 시간**: 무제한

### 포지션 청산 조건
1. **손절**: ROI -30% 도달시 자동 청산
2. **역신호**: 반대 진입 신호 발생시 청산 후 포지션 전환
3. **긴급청산**:
   - 롱 포지션: 1m/3m/5m 모두 120분간 연속 역배열시
   - 숏 포지션: 1m/3m/5m 모두 120분간 연속 정배열시

### 리스크 관리
- **손절**: -30%
- **익절**: 없음 (역신호까지 보유)
- **부분청산**: ROI 30%에서 30% 청산
- **트레일링 스톱**: ROI 30%에서 손익분기점 이동, 최고점에서 30% 트레일링
- **최대 보유시간**: 무제한

### 특수 기능
- **수동 포지션 추적**: 사용자가 수동으로 진입한 포지션도 자동 관리
- **텔레그램 알림**: 모든 주요 이벤트 실시간 알림
- **다중 타임프레임 분석**: 5개 타임프레임 동시 모니터링
- **AI 분석**: GPT-4o를 통한 선택적 시장 분석

## 🔧 설치 및 설정

### 1. 환경 변수 설정
`.env` 파일 생성:
```env
# Binance API (필수)
BINANCE_API_KEY="your_binance_api_key"
BINANCE_API_SECRET="your_binance_api_secret"

# 텔레그램 봇 (필수)
TELEGRAM_BOT_TOKEN="your_telegram_bot_token"
TELEGRAM_CHAT_ID="your_telegram_chat_id"

# OpenAI API (선택적 - AI 분석 사용시)
OPENAI_API_KEY="your_openai_api_key"

# 기타 API (선택적)
NEWSDATA_IO__KEY="your_newsdata_key"
COINGECKO_API_KEY="your_coingecko_key"
```

### 2. 필요 패키지 설치
```bash
pip install -r requirements.txt
```

또는 개별 설치:
```bash

pip install openai>=1.0.0 rich>=13.0.0 aiosqlite>=0.19.0 websockets>=11.0
```

### 3. 실행
```bash
python sol_main_v4.py
```
