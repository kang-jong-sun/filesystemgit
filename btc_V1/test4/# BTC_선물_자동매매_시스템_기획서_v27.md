## 1. 시스템 개요

### 1.1 핵심 전략

v27은 v25.2의 구조를 **완전 재설계**하여, 2,311,228개 조합을 3단계(Phase 1A/1B/2)에 걸쳐 전수 탐색한 결과물이다. 기존 v25.2의 MA 크로스 + MTF 지표 분리 전략을 발전시키면서, **거래 빈도 극대화 + 복리 효과 강화**에 중점을 두었다.

| 항목 | 값 |
|------|-----|
| 대상 | BTC/USDT 무기한 선물 |
| 초기 자본 | $3,000 USDT |
| 레버리지 | 10x |
| 기본 운용 비율 | 20~40% (모델별 차등) |
| DD 임계값 | -20% |
| 월간 손실 제한 | -20% |
| 수수료 | 0.04% (편도, Binance Taker) |
| 데이터 | 2020.01 ~ 2026.03 (655,399 5m bars, 약 75개월) |
| 데이터 가중치 | 2023~2026 높은 가중치, 2020~2022 참고 수준 |

### 1.2 설계 철학

1. **손익비 중심 구조**: 승률보다 손익비(Win/Loss Ratio)를 극대화하여 소수의 대형 수익 트레이드가 다수의 소형 손실을 상회하는 구조
2. **거래 빈도 극대화**: v25.2 대비 거래 수를 27~44배 증가시켜 복리 효과 극대화
3. **동적 진입**: 교차 후 0~60분 이내 유연한 진입 + 가격 허용 범위 적용
4. **멀티 타임프레임 독립 분리**: Cross, ADX, RSI를 각각 최적 TF에서 독립 계산
5. **리스크 적응**: DD 발생시 자동 마진 축소

### 1.3 최적화 규모

| 항목 | 값 |
|------|-----|
| Phase 1A (Cross Signal) | 124 조합 |
| Phase 1B (Filter 최적화) | 2,304,000 조합 → 112,246개 통과 |
| Phase 2 (Risk 최적화) | 7,104 조합 → 1,254개 통과 |
| 총 조합수 | 2,311,228 |
| 30회 반복 검증 | 모델별 30회 x 3모델 = 90회 |
| 엔진 | Numba JIT + Numpy 벡터화 |

## 2. v25.2 대비 진화

### 2.1 핵심 변화

| 구분 | v25.2 | v27 |
|------|-------|-----|
| 총 조합수 | 26,340 | 2,311,228 (87배) |
| 거래 빈도 | 15~63회 (6년) | 1,702~1,738회 (6년) |
| 진입 로직 | 교차 즉시 진입 | 교차 후 0~60분 유연 진입 |
| 가격 허용 범위 | 없음 | -0.5%~+0.5% |
| Cross Signal | EMA만 사용 | EMA, SMA, HMA 혼합 |
| ADX 분리 | 동일 TF 가능 | 독립 TF (5m~30m) |
| RSI 분리 | 동일 TF 가능 | 독립 TF (5m~15m) |
| MACD | 미사용 | 방향 필터로 추가 |
| 청산 방식 | SL+TSL+REV | SL+TSL+REV (비율 최적화) |
| PF 범위 | 7.58~17.20 | 1.21~1.24 (높은 거래 빈도) |
| MDD 범위 | 28.3~59.4% | 15.8~21.1% |

### 2.2 전략 방향 전환

**v25.2**: 소수 고확률 거래 → 높은 PF (10+), 낮은 거래 빈도 (15~63회)
**v27**: 다수 손익비 거래 → 안정적 PF (1.2+), 높은 거래 빈도 (1,700+회)

이 전환의 근거:
- 거래 빈도 증가 → 복리 효과 27배 이상 가속
- MDD 대폭 감소 (59.4% → 21.1%)
- 월별 손실 발생 빈도 분산 (한 달에 집중되지 않음)
- 심리적 인내 부담 감소 (v25.2 Model B는 17개월 무거래)
