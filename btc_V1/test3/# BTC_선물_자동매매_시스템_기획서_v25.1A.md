## 2. 전략 설정 상세

### 2.1 지표 구성

| 지표 | 파라미터 | 역할 |
|------|----------|------|
| **HMA (Hull Moving Average)** | 길이: 21 | 단기 추세 방향 (빠른 반응) |
| **EMA (Exponential MA)** | 길이: 250 | 장기 추세 기준선 |
| **ADX (Average Directional Index)** | 길이: 20, 임계값: >= 35 | 추세 강도 필터 |
| **RSI (Relative Strength Index)** | 길이: 14, 범위: 40~75 | 과매수/과매도 필터 |

### 2.2 진입 조건

#### 매수 (LONG) 진입
1. HMA(21) > EMA(250) (골든크로스 / 정배열)
2. ADX(20) >= 35 (강한 추세 확인)
3. 40 <= RSI(14) <= 75 (과매수 구간 제외)
4. 위 3개 조건 동시 충족 시 LONG 진입

#### 매도 (SHORT) 진입
1. HMA(21) < EMA(250) (데드크로스 / 역배열)
2. ADX(20) >= 35 (강한 추세 확인)
3. 40 <= RSI(14) <= 75 (과매도 구간 제외)
4. 위 3개 조건 동시 충족 시 SHORT 진입

### 2.3 청산 조건

| 조건 | 설명 |
|------|------|
| **손절 (SL)** | ROI -6% 도달 시 즉시 청산 |
| **트레일링 스톱 활성화** | ROI +7% 도달 시 |
| **트레일링 폭** | 최고점 대비 -3% 하락 시 청산 |
| **역신호 (REV)** | 반대 방향 신호 발생 시 즉시 청산 후 포지션 전환 |

## 3. 리스크 관리

### 3.1 포지션 사이징

| 항목 | 설정값 |
|------|--------|
| **레버리지** | 10배 |
| **기본 마진 비율** | 계좌 고정 1000 USDT |
| **축소 마진 비율** | 계좌 고정 1000 USDT (DD > 15% 시) |
| **최대 손실 한도 (ML)** | -20% (계좌 기준) |
| **마진 모드** | 격리마진 (ISOLATED) |

### 3.2 드로다운 관리

- **DD(Drawdown) > 15%**: 포지션 크기를 50% -> 25%로 축소
- **ML(Maximum Loss) -20%**: 거래 중지 후 수동 개입 필요
- **MDD 실적**: 최대 46.2% (백테스트 기준)

### 3.3 손절/익절 체계

```
진입 -> ROI 모니터링
  |
  +-- ROI <= -6%  -> 손절 (SL) 청산
  |
  +-- ROI >= +7%  -> 트레일링 스톱 활성화
  |                   최고점 추적 시작
  |                   최고점 - 3% 도달 시 청산 (TSL)
  |
  +-- 역신호 발생  -> 즉시 청산 (REV) + 반대 포지션 전환
```
