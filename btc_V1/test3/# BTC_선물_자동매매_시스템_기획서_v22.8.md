## 3. 진입/청산 로직

### 3.1 진입 조건

```
매수(LONG) 진입:
  1. EMA(100)이 EMA(600)을 상향 돌파 (골든크로스)
  2. 크로스 후 24봉(12시간) 이내
  3. ADX(20) >= 30
  4. ADX(20) > ADX(20)[6봉 전]  (ADX 상승 중)
  5. 35 <= RSI(14) <= 75
  6. |EMA(100) - EMA(600)| / EMA(600) >= 0.2%  (EMA 갭 필터)
  7. 직전 청산이 LONG이 아닌 경우 (동일방향 스킵)
  8. 일일 손실 -20% 미만이 아닌 경우

매도(SHORT) 진입:
  1. EMA(100)이 EMA(600)을 하향 돌파 (데드크로스)
  2~8. LONG과 동일 (방향만 반대)
```

### 3.2 청산 조건 (우선순위)

```
1순위: 손절 (SL)
  - LONG: 저가 <= 진입가 × (1 - 8%) → SL가에 청산
  - SHORT: 고가 >= 진입가 × (1 + 8%) → SL가에 청산

2순위: 트레일링 스톱 (TSL)
  - 활성화: 고점(LONG) 또는 저점(SHORT) 대비 +6% 이상 수익 도달
  - 활성화 후: 고점(LONG) 또는 저점(SHORT) 대비 -5% 하락 시 청산
  - SL가는 TSL 방향으로만 이동 (역행 불가)

3순위: 역신호 청산 (REV)
  - LONG 보유 중 EMA(100) < EMA(600) 전환 시 즉시 청산
  - SHORT 보유 중 EMA(100) > EMA(600) 전환 시 즉시 청산
```

### 3.3 EMA 갭 필터 상세

```
EMA 갭 = |EMA(100) - EMA(600)| / EMA(600) × 100

갭 < 0.2%  → 횡보 구간 판단 → 진입 차단
갭 >= 0.2% → 추세 구간 판단 → 진입 허용

[효과]
- 2025년 6~12월: EMA가 수렴한 횡보장에서 12회 허위 크로스 진입 → 갭 필터로 전부 차단
- REV 청산: 40건 → 26건 (불필요 진입 14건 제거)
- PF: 2.2 → 5.0 (127% 개선)
- 2025년 PnL: -$131,975 → +$306,641
```

## 4. 포지션 사이징 & 리스크 관리

### 4.1 포지션 크기

```
마진 = 1000 USDT (고정)
포지션 크기 = 마진 × 10 (레버리지)
수수료 = 포지션 크기 × 0.04% (진입 시 차감)

예시 (잔액 $100,000):
  마진 = $35,000
  포지션 = $10,000 (고정)
  수수료 = $140
  SL 1회 최대 손실 = $350,000 × 8% = $28,000 (고정 1000 USDT)
```

### 4.2 리스크 제어

| 항목 | 설정 | 설명 |
|------|------|------|
| SL 1회 최대 손실 | 고정 1000 USDT | M35% × 10x × 8% |
| 일일 최대 손실 | -20% | 일일 시작 잔액 대비 -20% 시 당일 거래 중단 |
| 교차마진 | CROSS | 잔액 전체가 마진으로 활용 → 강제청산 방지 |
| 동일방향 스킵 | YES | 연속 손실 방지 |

### 4.3 강제청산 리스크

```
교차마진 강제청산가:
  LONG: 진입가 × (1 - 잔액/포지션크기)
  → 잔액 $100,000 / 포지션 $350,000 = 28.6% 하락 시 강제청산
  → SL 8%가 먼저 발동하므로 강제청산 불가능 (이론상)
```

## 7. 지표 계산 공식

### 7.1 EMA (Exponential Moving Average)

```python
def ema(series, period):
    return series.ewm(span=period, adjust=False).mean()
```

### 7.2 ADX (Average Directional Index) - Wilder's Smoothing

```python
def adx(high, low, close, period=20):
    # +DM / -DM
    pdm = high.diff()
    mdm = -low.diff()
    pdm = pdm.where((pdm > mdm) & (pdm > 0), 0.0)
    mdm = mdm.where((mdm > pdm) & (mdm > 0), 0.0)

    # True Range
    tr = pd.concat([high-low, (high-close.shift()).abs(),
                     (low-close.shift()).abs()], axis=1).max(axis=1)

    # Wilder's Smoothing (alpha=1/period)
    atr = tr.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
    pdi = 100 * pdm.ewm(alpha=1/period, min_periods=period, adjust=False).mean() / atr
    mdi = 100 * mdm.ewm(alpha=1/period, min_periods=period, adjust=False).mean() / atr

    # DX → ADX
    dx = 100 * (pdi - mdi).abs() / (pdi + mdi).replace(0, 1e-10)
    return dx.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
```

### 7.3 RSI (Relative Strength Index)

```python
def rsi(series, period=14):
    delta = series.diff()
    gain = delta.where(delta > 0, 0.0)
    loss = (-delta).where(delta < 0, 0.0)
    avg_gain = gain.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
    avg_loss = loss.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
    return 100 - 100 / (1 + avg_gain / avg_loss.replace(0, 1e-10))
```

### 7.4 EMA 갭

```python
def ema_gap(ema_fast, ema_slow):
    return abs(ema_fast - ema_slow) / ema_slow * 100
    # >= 0.2% 이면 진입 허용
```
