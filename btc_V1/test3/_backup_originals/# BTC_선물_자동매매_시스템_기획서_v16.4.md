## 2. v13.5~v16.0 전 버전 대비 혁신

### 2.1 ADX Wilder's Smoothing 수정

기존 모든 버전(v13.5~v16.0)의 ADX 계산에서 **DX->ADX 스무딩 공식 오류 발견 및 수정**:

```
[기존 오류]
ADX 첫값 = SUM(DX) (합계)
이후값 = prev * (N-1)/N + current_DX (전체 DX 반영)
-> ADX 값이 ~N배 과대 산출

[v16.4 수정]
ADX 첫값 = MEAN(DX) (평균, 올바른 Wilder's)
이후값 = (prev * (N-1) + current_DX) / N (1/N 비율 반영)
-> ADX 0~100 정상 범위
```

이 수정으로 ADX 필터가 정상 작동하여 신호 품질이 극적으로 향상.

## 6. 인디케이터 계산식

### 6.1 WMA(3) - Weighted Moving Average

```
WMA(3) = (Close[n-2]*1 + Close[n-1]*2 + Close[n]*3) / 6
가중치: 최근봉에 최대 가중 (1:2:3)
특징: EMA(3)보다 반응 빠름, PF 4.7배 우수
```

### 6.2 EMA(200) - Exponential Moving Average

```
EMA(200) = Close * (2/201) + EMA_prev * (199/201)
감쇠율: 0.00995 (매우 느림, ~4.2일 실효 기간)
```

### 6.3 ADX(20) - Wilder's Smoothing (수정됨)

```
[Step 1-3] TR, +DM, -DM 계산
[Step 4] Wilder's Smoothing (첫값=합계, 이후 prev*(N-1)/N + current)
[Step 5] +DI, -DI = Smoothed_DM / Smoothed_TR * 100
[Step 6] DX = |+DI - -DI| / (+DI + -DI) * 100
[Step 7] ADX = 첫값 MEAN(DX[20:40]), 이후 (prev*(N-1) + current)/N
         *** v16.4 핵심 수정: Step 7의 /N 추가 ***
```

### 6.4 RSI(14) - Wilder's Smoothing

```
첫값: avg_gain = SMA(gain, 14), avg_loss = SMA(loss, 14)
이후: avg_gain = (prev*13 + gain) / 14
RSI = 100 - 100/(1 + avg_gain/avg_loss)
```
