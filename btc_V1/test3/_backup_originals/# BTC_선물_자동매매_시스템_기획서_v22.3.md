## 12. WMA(250) 구현

```python
def calc_wma(close, period=250):
    """가중이동평균: 최근 가격에 더 큰 가중치"""
    weights = np.arange(1, period + 1, dtype=float)
    weight_sum = weights.sum()
    result = np.full(len(close), np.nan)
    for i in range(period - 1, len(close)):
        result[i] = np.dot(close[i - period + 1:i + 1], weights) / weight_sum
    return result
```

### EMA(200) vs WMA(250) 차이

| 항목 | EMA(200) | WMA(250) |
|------|----------|----------|
| 가중 방식 | 지수 감소 | 선형 증가 |
| 최근 가격 반응 | 빠름 | 중간 |
| 노이즈 필터 | 보통 | 강함 |
| 크로스 빈도 | 105 (ADX35) | 174 (ADX25) |
| 최적 ADX | >=35 | **>=25** |
