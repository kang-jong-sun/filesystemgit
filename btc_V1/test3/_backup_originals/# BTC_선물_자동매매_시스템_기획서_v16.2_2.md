## 6. 엔진 B 청산 및 리스크 관리

### 6.1 ATR 기반 동적 손절

| 파라미터 | 엔진 B |
|----------|--------|
| ATR 배수 | 2.0~3.0 |
| 예상 SL 범위 | 2~6% |

```
Long SL = Entry_Price - ATR(14, 15m) × ATR_SL_Multiplier
Short SL = Entry_Price + ATR(14, 15m) × ATR_SL_Multiplier

최소 SL: 1% | 최대 SL: 8% (레버리지 10x 기준)
```

### 6.2 분할 청산 구조

```
TP1: ATR × 3~4 도달 → 30% 물량 청산 → SL을 손익분기점으로
TP2: ATR × 5~6 도달 → 30% 물량 청산 → SL을 TP1 수준으로
TP3: 트레일링 스톱으로 나머지 40% 관리
```

### 6.3 트레일링 스톱

```
Trail_Distance = ATR(14, 15m) × Trail_ATR_Multiplier (2.0~3.0)
Long: Trail_SL = Highest_High - Trail_Distance
Short: Trail_SL = Lowest_Low + Trail_Distance

활성화 기준: ROI ≥ 5~8% (최적화 대상)
단계별:
  ROI 5%+ → Trail 5%
  ROI 10%+ → Trail 4%
  ROI 15%+ → Trail 3%
  ROI 20%+ → Trail 2.5%
```

### 6.4 역신호 청산

```
현재 LONG 보유 중 + SHORT 진입 조건 발생:
  → LONG 전량 청산 (시장가) → SHORT 즉시 진입

기존 포지션 ROI > 10%인 경우: 50% 부분 청산 후 나머지 트레일링
```
