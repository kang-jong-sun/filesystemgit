### HMA(21) 계산

```python
def hma(series, period=21):
    """Hull Moving Average - 노이즈 제거 + 빠른 반응"""
    half = max(int(period / 2), 1)
    sqrt_p = max(int(np.sqrt(period)), 1)
    wma_half = wma(series, half)
    wma_full = wma(series, period)
    return wma(2 * wma_half - wma_full, sqrt_p)
```

> HMA는 WMA 기반으로 **지연을 줄이면서 노이즈를 제거**하는 고급 이동평균.
> EMA(21) 대비 크로스 품질이 높아 잡 신호 50%+ 감소.
