```

          |
          v

          |
          v

          |
          v

```


## 14. 지표 구현

### WMA (Weighted Moving Average)

```python
def calc_wma(close, period):
    weights = np.arange(1, period + 1, dtype=float)
    weight_sum = weights.sum()
    result = np.full(len(close), np.nan)
    for i in range(period - 1, len(close)):
        result[i] = np.dot(close[i - period + 1:i + 1], weights) / weight_sum
    return result
```

### EMA

```python
ema200 = close.ewm(span=200, adjust=False).mean()
```

### ADX (Wilder's Smoothing, period=20)

```python
adx = dx.ewm(alpha=1/20, min_periods=20, adjust=False).mean()

```

### RSI (Wilder's Smoothing)

```python
gain_avg = gain.ewm(alpha=1/14, min_periods=14, adjust=False).mean()
loss_avg = loss.ewm(alpha=1/14, min_periods=14, adjust=False).mean()

```
