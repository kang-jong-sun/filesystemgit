## 6. 청산 로직

| 순위 | 트리거 | 방식 |
|------|--------|------|
| 1순위 | 기본 SL **-7%** | 바이낸스 서버 Stop Market |
| 2순위 | 트레일링 (수익 **+6%** 후 고점 **-3%**) | 봇 30분봉 종가 |
| 3순위 | 역신호 (EMA 반대 크로스 + ADX>=35 + RSI 30-65) | 봇 30분봉 |

### SL

```
롱: SL = 진입가 x 0.93
숏: SL = 진입가 x 1.07
```

### 트레일링 SL

| 항목 | 설정 |
|------|------|
| 활성화 | 수익 **+6%** 도달 |
| 폭 | 고점/저점 대비 **-3%** |
| 추적 | 캔들 high/low |
| 판단 | 30분봉 종가 |

**트레일링 예시 (롱):**
```
진입가: $100,000
+6% 도달 ($106,000) -> 트레일링 활성화
고점 $110,000 -> TSL = $110,000 x 0.97 = $106,700
고점 $115,000 -> TSL = $115,000 x 0.97 = $111,550
종가 $111,000 (<= $111,550) -> 청산 (수익 +11%)
```

### 손익비 구조

| 항목 | 값 |
|------|-----|
| 평균 승 | +8.09% |
| 평균 패 | -1.81% |
| **손익비** | **4.47:1** |
| 의미 | 1회 승리 = 4.47회 손실 상쇄 |

> 승률 45.7%에서도 높은 수익이 가능한 이유:
> - 수익 거래는 트레일링으로 크게 키움 (+8.09%)
> - 손실 거래는 SL로 빠르게 차단 (-1.81%)
> - 4.47:1 손익비로 45% 승률이면 기대값 양(+)

adx = dx.ewm(alpha=1/14, min_periods=14, adjust=False).mean()

```python
gain_avg = gain.ewm(alpha=1/14, min_periods=14, adjust=False).mean()
loss_avg = loss.ewm(alpha=1/14, min_periods=14, adjust=False).mean()
rsi = 100 - 100 / (1 + gain_avg / loss_avg)
```
