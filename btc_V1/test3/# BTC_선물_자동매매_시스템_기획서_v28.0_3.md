## VWMA(2) + VWMA(300) 구조의 특성

Tier 3은 Fast와 Slow 모두 VWMA 계열:

| MA | Tier 1 | Tier 2 | Tier 3 |
|----|--------|--------|--------|
| Fast | WMA(5) | HMA(14) | **VWMA(2)** |
| Slow | VWMA(300) | VWMA(300) | **VWMA(300)** |

**VWMA(2) Fast MA의 특성**:
- 매우 짧은 기간(2봉) → 극도로 빠른 반응
- 거래량 가중으로 노이즈 일부 필터링
- 즉시 진입(0캔들)과 조합하여 추세 초기 포착 극대화

**ADX(14) >= 45의 역할**:
- Tier 1/2의 ADX(20) >= 35보다 훨씬 엄격
- 강력한 추세에서만 빠른 진입 허용
- 낮은 승률(36.7%)을 높은 손익비(9.68:1)로 보완

## 구현 상세

### VWMA 구현 (Fast + Slow)

```python
def calc_vwma(close, volume, period):
    """Volume Weighted Moving Average"""
    vwma = (close * volume).rolling(period).sum() / volume.rolling(period).sum()
    return vwma

# Tier 3 사용
fast_ma = calc_vwma(close, volume, period=2)     # VWMA(2)
slow_ma = calc_vwma(close, volume, period=300)   # VWMA(300)
```
