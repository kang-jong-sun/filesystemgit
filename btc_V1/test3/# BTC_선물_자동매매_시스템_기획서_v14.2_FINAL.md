```

[작동 방식]
  롱:
    - 캔들 high 기준으로 trail_high를 지속 갱신
    - new_sl = trail_high × (1 - 0.01) = trail_high × 0.99
    - new_sl > 현재 SL이면 SL 상향 조정
    - 캔들 low가 SL 이하 → 청산

  숏:
    - 캔들 low 기준으로 trail_low를 지속 갱신
    - new_sl = trail_low × (1 + 0.01) = trail_low × 1.01
    - new_sl < 현재 SL이면 SL 하향 조정
    - 캔들 high가 SL 이상 → 청산

[예시 - 롱 포지션]
  진입가: $70,000
  +10% → $77,000 도달 → 트레일링 활성화

  고점 $85,000 → SL = $85,000 × 0.99 = $84,150
  고점 $90,000 → SL = $90,000 × 0.99 = $89,100
  $89,100 하회 시 → 청산 (수익 약 +27.3%)
```

## 5. HMA (Hull Moving Average) 상세

### 5-1. HMA 공식

```
HMA(n) = WMA( 2 × WMA(n/2) - WMA(n), √n )

HMA(7) 계산:
  Step 1: WMA(3) = WMA(close, 3)       // 반 길이
  Step 2: WMA(7) = WMA(close, 7)       // 전체 길이
  Step 3: diff = 2 × WMA(3) - WMA(7)   // 차이
  Step 4: HMA(7) = WMA(diff, 2)        // √7 ≈ 2.6 → 2
```

### 5-2. HMA vs EMA 비교

| 특성 | EMA | HMA |
|------|-----|-----|
| 지연(Lag) | 보통 | **50% 이상 적음** |
| 반전 감지 | 느림 | **빠름** |
| 노이즈 | 많음 | **자체 스무딩** |
| 허위 크로스 | 많음 | **적음** |
| 계산 복잡도 | 낮음 | 보통 |

### 7-2. -1% 트레일링의 논리

```
핵심 원리: "큰 수익은 크게, 보호는 빠르게"

  - 승률 38%로 손실 거래가 더 많음
  - 따라서 수익 거래에서 최대한 많이 확보해야 함
  - +10% 이상 수익이 발생한 것은 강한 추세 존재를 의미
  - 그 시점부터 -1%로 타이트하게 추적하면:
    → 추세가 계속되면 수익 극대화
    → 반전 시 즉시 수익 확보 (고점 대비 -1%만 손실)

  결과: 손익비 극대화 → 낮은 승률에도 높은 수익
```

## 13. 인디케이터 구현 공식

### 13-1. HMA (Hull Moving Average)

```python
def hma(series, period):
    half_period = max(int(period / 2), 1)      # 7 → 3
    sqrt_period = max(int(sqrt(period)), 1)     # 7 → 2
    wma_half = wma(series, half_period)         # WMA(close, 3)
    wma_full = wma(series, period)              # WMA(close, 7)
    diff = 2 * wma_half - wma_full              # 차이
    return wma(diff, sqrt_period)               # WMA(diff, 2)
```

### 13-2. EMA (Exponential Moving Average)

```python
def ema(series, period):
    return series.ewm(span=period, adjust=False).mean()
```

### 13-3. ADX (Wilder's Smoothing)

```python
def adx(high, low, close, period=20):
    # +DM, -DM 계산
    plus_dm = high.diff().where((high.diff() > -low.diff()) & (high.diff() > 0), 0)
    minus_dm = (-low.diff()).where((-low.diff() > high.diff()) & (-low.diff() > 0), 0)

    # True Range
    tr = max(high-low, abs(high-close.shift()), abs(low-close.shift()))

    # Wilder's Smoothing (alpha = 1/period)
    atr = tr.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
    plus_di = 100 * plus_dm.ewm(alpha=1/period, ...).mean() / atr
    minus_di = 100 * minus_dm.ewm(alpha=1/period, ...).mean() / atr

    # DX → ADX
    dx = 100 * abs(plus_di - minus_di) / (plus_di + minus_di)
    return dx.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
```

### 13-4. RSI (Wilder's Smoothing)

```python
def rsi(series, period=14):
    delta = series.diff()
    gain = delta.where(delta > 0, 0)
    loss = (-delta).where(delta < 0, 0)
    avg_gain = gain.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
    avg_loss = loss.ewm(alpha=1/period, min_periods=period, adjust=False).mean()
    rs = avg_gain / avg_loss
    return 100 - (100 / (1 + rs))
```
