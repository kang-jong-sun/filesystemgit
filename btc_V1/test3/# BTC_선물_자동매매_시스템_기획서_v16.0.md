```

WMA(3)의 특성:
  1. 노이즈 반응이 EMA(3)보다 느림 → 허위 크로스 감소
  2. 추세 전환 시 EMA보다 약간 지연 → 확인 후 진입 효과
  3. 결과: 동일 ADX/RSI 조건에서 잡 신호 50%+ 제거

증거 (Phase 1 30+ 거래 PF Top):
  #1 WMA(3/200) PF 10.72 → #2 WMA(5/150) PF 9.75
  #3 WMA(3/200) PF 7.38 → EMA(3/200)는 Phase 1에서 PF 2.30

  → WMA(3)가 EMA(3) 대비 PF 4.7배 향상!
```

```
[30분봉 WMA(3) / EMA(200) 크로스]     ← WMA가 핵심!
          |
          v

          |
          v
[RSI(14) 35 ~ 65] → 극단 회피
          |
          v
[시장가 진입 (잔액 50% 증거금 x 10배)]
          |
          v
[역신호 / TSL(+4% 활성화, -3% 추적) / SL(-8%)]
```

### WMA(3) 계산

```python
def wma(series, period=3):
    """Weighted Moving Average"""
    weights = np.arange(1, period + 1, dtype=float)  # [1, 2, 3]
    return series.rolling(period).apply(
        lambda x: np.dot(x, weights) / weights.sum(), raw=True
    )

# WMA(3): 최근 3개 종가에 가중치 1:2:3
# 예: close = [100, 102, 105]
# WMA = (100*1 + 102*2 + 105*3) / (1+2+3) = 619/6 = 103.17
```
