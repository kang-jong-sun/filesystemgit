### 진입/청산 로직

```python
# 매수(LONG) 진입:
if WMA(3) > EMA(200)               # 골든 크로스 발생
   and WMA(3)_prev <= EMA(200)_prev  # 이전 봉에서는 아래
   and ADX(20) >= 35                 # 추세 강도 충분
   and 35 <= RSI(14) <= 65           # 과열 아님
   and 월손실제한 미발동:
   → LONG 진입 (시장가)

# 매도(SHORT) 진입:
if WMA(3) < EMA(200)               # 데드 크로스 발생
   and WMA(3)_prev >= EMA(200)_prev
   and ADX(20) >= 35
   and 35 <= RSI(14) <= 65
   and 월손실제한 미발동:
   → SHORT 진입 (시장가)
```

### 인디케이터 계산식

```
WMA(3):
  Weight = [1, 2, 3]
  WMA = (P[0]*1 + P[1]*2 + P[2]*3) / 6

EMA(200):
  α = 2 / (200 + 1) = 0.00995
  EMA(n) = Price × α + EMA_prev × (1-α)

ADX(20) — Wilder's Smoothing:
  첫 값: DX의 MEAN (SUM 아님 - v16.4 버그 수정)

RSI(14):
  RSI = 100 - 100 / (1 + RS)
  RS = Avg_Gain / Avg_Loss  [Wilder's Smoothing, α=1/14]
```
