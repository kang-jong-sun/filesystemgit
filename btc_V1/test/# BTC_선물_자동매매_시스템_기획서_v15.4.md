## 4. 전략 구조

```
[30분봉 EMA(3) / EMA(200) 크로스]
          |
          v
[ADX(14) >= 35] -> 강한 추세 확인
          |
          v
[RSI(14) 30 ~ 65] -> 극단 회피
          |
          v
[보호] 월간 손실 -30% 초과? -> 이번 달 중단
          | NO
          v
[시장가 진입 (잔액 40% 증거금)]
          |
          v
[역신호 / 트레일링 SL / 기본 SL] -> [시장가 청산]
```

## 6. 포지션 관리

| 항목 | 설정 |
|------|------|
| 마진 모드 | 격리마진 (ISOLATED) |
| 레버리지 | 10배 |
| 증거금 | 잔액의 **40%** |
| 포지션 | 잔액의 **400%** |

```
$3,000 계좌:
증거금:         $3,000 x 40% = $1,200
포지션:         $1,200 x 10배 = $12,000
SL 시 손실:     $12,000 x 7% = $840 (잔액의 28%)
강제청산 시 손실: $1,200 (잔액의 40%)
강제청산 여유:   10% - 7% = 3%p
```

## 8. 보호 메커니즘

| -7% | 20% | 14% | 3%p |
| -7% | 25% | 17.5% | 3%p |
| -7% | 30% | 21% | 3%p |
| **-7%** | **40%** | **28%** | **3%p** |

> 강제청산 여유는 마진과 무관하게 항상 3%p. SL 1회 손실이 잔액의 28%로 크지만, **75개월 중 SL 11회 vs TSL 47회**로 수익 거래가 압도적.
