### 3.2 SL/TSL 변경 근거

```
[기존 v22.8] SL8% TA6/TSL5
- SL 1회 잔액 손실: 8% × 35% × 10 = 28%
- TSL 활성화: +6%에서 활성 → 조기 활성 → 소폭 수익에서 청산
- TSL 폭: -5% → 고점 대비 빠르게 청산 → 큰 추세 놓침

[v32.1] SL3% TA12/TSL10
- SL 1회 잔액 손실: 3% × 35% × 10 = 10.5%
- TSL 활성화: +12%에서 활성 → 충분한 수익 확보 후 활성화
- TSL 폭: -10% → 고점 대비 넉넉한 여유 → 큰 추세 끝까지 보유

효과:
- SL이 빈번하지만(31회) 1회 손실이 작음 → 잔액 보존
- TSL이 느슨하여 수익 거래의 평균 ROI 증가
- "많이 지고 크게 이기는" 추세추종 본질에 부합
```

## 4. 진입/청산 로직

### 4.1 진입 조건

```
매수(LONG) 진입:
  1. Fast MA가 Slow MA를 상향 돌파 (골든크로스)
  2. 크로스 후 24봉(12시간) 이내
  3. ADX(20) >= 30
  4. ADX(20) > ADX(20)[6봉 전] (ADX 상승 중)
  5. 35 <= RSI(14) <= 75
  6. |Fast MA - Slow MA| / Slow MA >= 0.2% (EMA 갭 필터)
  7. 직전 청산이 LONG이 아닌 경우 (동일방향 스킵)
  8. 일일 손실 -20% 미만이 아닌 경우

매도(SHORT) 진입: 조건 반대
```

### 4.2 청산 조건 (우선순위)

```
1순위: 손절 (SL) -3%
  - LONG: 저가 <= 진입가 × 0.97
  - SHORT: 고가 >= 진입가 × 1.03
  → SL 1회 잔액 손실 = 10.5%

2순위: 트레일링 스톱 (TSL) +12%/-10%
  - 활성화: ROI +12% 도달
  - 청산: 고점(LONG) 또는 저점(SHORT) 대비 -10% 하락
  → 큰 추세를 끝까지 보유하는 핵심 장치

3순위: 역신호 청산 (REV)
  - LONG 보유 중 Fast MA < Slow MA 전환 시 즉시 청산
  - SHORT 보유 중 Fast MA > Slow MA 전환 시 즉시 청산
```

## 6. 리스크 분석

### 6.1 SL 3% 리스크

```
SL 1회 잔액 손실 = 3% × 35% × 10 = 10.5%
연속 SL 시:
  2연속: 잔액 × 0.895² = 80.1% (19.9% 감소)
  3연속: 잔액 × 0.895³ = 71.7% (28.3% 감소)
  5연속: 잔액 × 0.895⁵ = 57.4% (42.6% 감소)

실제 발생:
  A안: 2022년 8연속 SL (#26~#34) → 잔액 $260K → $114K (56.2% 감소)
  → 이후 #36(+$137K)으로 즉시 복구

교훈: 연속 SL 후 1건의 대형 수익이 전부 복구
```

### 6.2 MDD 분석

```
A안 MDD 56.8%: 2022년 9~12월 연속 SL로 형성 (8연속 SL)
  - $263,602 (고점) → $113,951 (저점)
  - 이후 2023년 #36~#38에서 완전 복구

B안 MDD 43.5%: 2022년 동일 구간이지만 SL이 덜 치명적
  - SMA(750)이 더 느려서 일부 허위 크로스 차단

### 8.3 공통 주의사항

```
1. SL 3%는 빈번한 손절 발생 (70건 중 31~34회)
   → 심리적 부담 있음. 시스템 신뢰하고 자동 운영 필수

2. TSL 12/10은 수익이 +12%까지 올라야 활성화
   → 소폭 수익(+3~10%)은 REV로 청산됨
   → REV 청산이 26건으로 많음

3. 교차마진이므로 강제청산 리스크는 낮지만
   → SL 3%가 먼저 발동하므로 이론상 강제청산 불가

4. EMA 갭 필터 0.2%로 횡보장 진입 차단
   → 그래도 2022년 같은 장기 횡보에서는 SL 누적 가능
