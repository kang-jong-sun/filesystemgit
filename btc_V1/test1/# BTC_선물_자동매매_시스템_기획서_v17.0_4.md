### 인디케이터 계산식

```
WMA(3):
  Weight = [1, 2, 3]
  WMA = (P[0]*1 + P[1]*2 + P[2]*3) / 6

EMA(200):
  α = 2 / (200 + 1) = 0.00995
  EMA(n) = Price × α + EMA_prev × (1-α)

ADX(20) — Wilder's Smoothing (v16.4 수정):
  첫 값: DX의 MEAN (SUM 아님)

RSI(14):
  범위: 30~70 (Engine B/C의 35~65보다 넓음)
  RSI = 100 - 100 / (1 + RS)
  RS = Avg_Gain / Avg_Loss  [Wilder's Smoothing, α=1/14]
```

### 강제청산 안전 마진 (M50%)

```
포지션 규모: 잔액 × M50% × L10x = 잔액 × 5배
SL 발동가: 진입가 × (1 ± 8%)
FL 발동가: 진입가 × (1 ± 10%)  [격리마진]

SL → FL 안전 거리: 2%p
→ 75개월 검증: FL 0회 (SL도 0회)
→ 5캔들 지연 + REV 청산이 모든 손실을 사전 차단
```
