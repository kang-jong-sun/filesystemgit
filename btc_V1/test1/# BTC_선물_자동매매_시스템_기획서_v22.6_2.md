## 멀티TF 순차진입 구조

```
교차마진 단일 계좌 $5,000

포지션 없을 때 순차적으로 신호 확인:
  1순위: [TF2] 30m WMA(3)/EMA(200) ADX(20)>=35 RSI 35~60 크로스  ← 이 문서
  2순위: 15m EMA(7)/EMA(250) ADX(14)>=45 RSI 35~75 크로스
  3순위: 60m EMA(3)/EMA(100) ADX(20)>=30 RSI 30~70 크로스

→ 어떤 TF에서든 신호 발생 시 단일 포지션 진입
→ TSL 청산 후 3캔들 뒤 추세 지속 시 리엔트리

결과: 18거래(단일TF) → 115거래(멀티TF+리엔트리) = 6.4배 증가
```

## TF2 (30m) 전략 설정

### 핵심 파라미터

```python
# ═══ TF2 30m 신호 (1순위) ═══
TF2_TIMEFRAME = '30m'
TF2_MA_FAST   = ('WMA', 3)
TF2_MA_SLOW   = ('EMA', 200)
TF2_ADX       = (20, 35)    # ADX period=20, threshold=35
TF2_RSI       = (35, 60)    # RSI min=35, max=60
```

### TF2 특성 비교

| 항목 | TF1 (15m) | TF2 (30m) | TF3 (60m) |
|------|-----------|-----------|-----------|
| MA Fast | EMA(7) | **WMA(3)** | EMA(3) |
| MA Slow | EMA(250) | **EMA(200)** | EMA(100) |
| ADX Period | 14 | **20** | 20 |
| ADX 임계값 | 45 | **35** | 30 |
| RSI 범위 | 35~75 | **35~60** | 30~70 |
| 우선순위 | 2순위 | **1순위** | 3순위 |
| 특징 | 높은 ADX 필터 | **핵심 주도 TF** | 최하위 ADX 필터 |

**TF2 (30m)의 RSI 상한 60은 세 TF 중 가장 엄격합니다 — 과열 구간 진입 강력 차단.**

## TF2 (30m) 진입 조건

### 진입 로직

```
[TF2 30m 신호 발생 조건 — 1순위]

1. 현재 포지션 없음 (항상 먼저 확인)
2. 30분봉 기준:
   - WMA(3) > EMA(200) 골든크로스 → LONG
   - WMA(3) < EMA(200) 데드크로스 → SHORT
3. ADX(20) >= 35
4. RSI 35~60 범위 내 (가장 좁은 RSI 범위)
5. 동일방향 스킵 (SKIP_SAME_DIR=True)
6. 월간 손실 한도 미도달
```

### TF2 RSI 35~60 특성

RSI 상한 60은 세 TF 중 가장 낮습니다:
- 이미 과열된 구간 (RSI 60+)에서 신규 진입 방지
- 추세 초기~중간 단계에서만 진입
- 허위 신호 필터링 효과 강화

## TF2 (30m) 청산 조건

### 청산 우선순위 (공통)

```
1순위: SL 발동 (-7%)
2순위: TSL 발동 (+5% 도달 후 최고점 -3%)
3순위: 역신호 발생 (반대 방향 크로스 + 조건 충족)
```

### 리엔트리

```
TSL 청산 후 3캔들(90분) 뒤:
  - 30m 추세 동일 방향 지속 중
  - ADX(20) >= 35 재충족
  - RSI 35~60 재충족
  → 재진입 허용
```
