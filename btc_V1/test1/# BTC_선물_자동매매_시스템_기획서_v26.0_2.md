### ADX Wilder's Smoothing 필수 공식

```python
def wilder_smooth(values, period):
    result = np.full_like(values, np.nan)
    result[period - 1] = np.mean(values[:period])  # seed = 평균 (SUM 아님!)
    for i in range(period, len(values)):
        result[i] = (result[i-1] * (period - 1) + values[i]) / period
    return result
```

> **ADX rolling().mean() 또는 pandas ewm() 사용 금지. Wilder 직접 구현 필수.**

### WMA(3) vs EMA(3) 비교 (Track B 채택 근거)

```
WMA(3): 가중이동평균, 가중치 [3, 2, 1] / 6 = 최근값 50% 가중
EMA(3): 지수이동평균, alpha=2/(3+1)=0.5, 지수적 감쇠

WMA(3)의 장점:
  1. 노이즈 반응이 EMA(3)보다 느림 → 허위 크로스 50%+ 감소
  2. 추세 전환시 EMA보다 약간 지연 → 확인 후 진입 효과
  3. v16.0 검증: WMA(3) PF 10.72 vs EMA(3) PF 2.30

WMA(3) 공식: WMA = (3*P[0] + 2*P[-1] + 1*P[-2]) / 6
```

## 지표 계산 공식

### WMA

```python
def wma(series, period):
    """
    WMA(3): weights = [3, 2, 1], divisor = 6
    WMA = (3 * Close[0] + 2 * Close[-1] + 1 * Close[-2]) / 6
    """
    weights = np.arange(1, period + 1)
    return series.rolling(period).apply(
        lambda x: np.dot(x, weights) / weights.sum(), raw=True
    )
```

### RSI (Wilder's Smoothing)

```python
def calculate_rsi(close, period=14):
    delta = close.diff()
    gain = delta.where(delta > 0, 0)
    loss = (-delta).where(delta < 0, 0)
    avg_gain = wilder_smooth(gain.values, period)
    avg_loss = wilder_smooth(loss.values, period)
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    return rsi
# RSI 30-70 필터: 추세 초중기 안전 진입 구간
```
