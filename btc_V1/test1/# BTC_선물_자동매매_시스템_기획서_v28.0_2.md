## 구현 상세

### HMA 구현

```python
def calc_hma(data, period):
    """Hull Moving Average = WMA(2*WMA(n/2) - WMA(n), sqrt(n))"""
    half = max(int(period / 2), 1)
    sqrt_p = max(int(np.sqrt(period)), 1)
    wma_half = calc_wma(data, half)
    wma_full = calc_wma(data, period)
    diff = 2 * wma_half - wma_full
    return calc_wma(diff, sqrt_p)
```

### VWMA 구현

```python
def calc_vwma(close, volume, period):
    """Volume Weighted Moving Average"""
    vwma = (close * volume).rolling(period).sum() / volume.rolling(period).sum()
    return vwma
```
