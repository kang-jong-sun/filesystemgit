## 2. 거래수가 거의 같은 이유

### 핵심 답변: 전략 규칙이 극도로 명확하기 때문

v14.4 전략의 진입 조건은 **3가지 이진 필터**로 구성되어 있다:

```
골든크로스 + ADX >= 35 + RSI 30~65 → LONG
데드크로스 + ADX >= 35 + RSI 30~65 → SHORT
하나라도 미충족 → HOLD
```

이 조건은 해석의 여지가 거의 없다. AI에게 "캔들 패턴으로 추세 강도 추가 확인"이라는
재량권을 부여했지만, 대부분의 크로스 시점에서 ADX와 RSI가 기준을 **명확히 충족하거나
명확히 미달**했기 때문에 AI도 동일한 판단을 내렸다.

### Action 분포 비교

